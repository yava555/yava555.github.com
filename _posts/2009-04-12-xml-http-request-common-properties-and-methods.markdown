XMLHttpRequest对象常用属性和方法：
<ul>
	<li>open()：建立到服务器的新请求。</li>
	<li>send()：向服务器发送请求。</li>
	<li>abort()：退出当前请求。</li>
	<li>onreadystatechange：指定回调方法。</li>
	<li>readyState：提供当前 HTML 的就绪状态。</li>
	<li>status：请求响应状态。</li>
	<li>responseText：服务器返回的请求响应文本。</li>
</ul>
